3. Cách sử dụng

Liên kết file từ thư viện

Dán đoạn mã này để link file js vào trước thẻ đóng body của file HTML 

<script src="https://cdn.jsdelivr.net/gh/dinhluckyboy/switchy@version/switchy.min.js"></script>

Khởi tạo Tabs

Đây là cấu trúc HTML cơ bản để thiết lập Switchy

<!-- tabs  -->
            <ul id="demo-tabs">
                   <li><a href="#demo1">tab 1</a></li>
                   <li><a href="#demo2">tab 2</a></li>
            </ul>
<!-- panel   -->
              <p id="demo1">Content tab 1</p>
              <p id="demo2">Content tab 2</p>

Đây là cấu trúc JS cơ bản để khởi tạo Switchy

 // demo 
      const tabs = new Switchy("#demo-tabs", {
        classNameActive: "switchy--active", // css class applied to the active tab
        remember: false, // remember tabs 
        onChange: animatePanels, // function after tab active   
      });
      function animatePanels() {
        //something handle...
      }
      animatePanels();

Using tabs methods

// Switch to a specific tab
      tabs.switch("#demo2");

// Destroy the tab instance
      tabs.destroy();

Chú thích về các thuộc tính khởi tạo

const tabs = new Switchy(selector, options);

selector là string chứa id tabs

option là object chứa các giá trị classNameActive, remember, onChange

  • classNameActive có giá trị mặc định là chuỗi "switchy--active"có thể thay đổi được, đây là css class của tab đang được kích hoạt
  • remember có giá trị mặc định là false khi là false tab đang kích hoạt sẽ không được lưu khi tải lại trình duyệt và ngược lại
  • onChange là function được gọi sau khi chuyển tab giúp bạn dễ dàng tuỳ chỉnh phát triển thêm các tính năng mới, giá trị mặc định là null

4. API

Method Description
switch(input) Dùng để chuyển đến một tab cụ thể, input có thể là selectorPanel ("#demo2") hoặc DOM element
destroy() Loại bỏ toàn bộ tính năng của Switchy, khôi phục HMTL gốc, huỷ liên kết các sự kiện
